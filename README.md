# BayadMuna

This project was built using the following version of AndroidStudio:
- Android Studio version: Android Studio Otter | 2025.2.1 Patch
- Gradle Wrapper: gradle-8.13-bin.zip

All the other project setup have been committed for your reference.

If you wish to build this on your end, you need to setup your Firebase Console as this uses Firebase as a means to register and authenticate logins. A helpful guide can be found here: https://youtu.be/QAKq8UBv4GI?si=3HF3OEBL-cs5gMmY

Primarly showcases the following:
- [DataBinding](https://developer.android.com/topic/libraries/data-binding): To showcase linking between UI (xml) and data sources (ViewModels).
= [Firebase Authentication](https://firebase.google.com/docs/auth): For authentication.
- [Hilt](https://developer.android.com/training/dependency-injection/hilt-android): To showcase skill in dependency injection.
- [OkHttp](https://square.github.io/okhttp/): To simulate API requests from a "server".
- [Mockk](https://mockk.io) and [JUnit5](https://www.baeldung.com/junit-5): (Pending) For unit testing.
